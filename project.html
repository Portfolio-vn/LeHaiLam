Project brief

Goal: Build a practical sensing device for lake/water quality monitoring.

Output: PCB design, working prototype, demo video, report, and technical evidence.

Focus: Hardware integration, signal measurement, data transmission, and engineering documentation.

Problem & solution

From raw sensor readings to a readable monitoring workflow.

Water quality can change quickly because of temperature shifts, dissolved solids, pollution sources, and biological activity. The project turns these variables into readable digital signals and sends them to a monitoring dashboard.

The core engineering challenge is not only connecting sensors, but designing a clear path from measurement to transmission: sensor probe → ESP32 ADC / 1-Wire → data processing → ThingSpeak / communication module → evidence and visualization.
